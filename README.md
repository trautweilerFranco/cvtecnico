# CV Web - Franco Trautweiler

Este proyecto es un CV técnico web. Para su desarrollo, opté por utilizar herramientas modernas del mercado actual, garantizando como resultado final un sitio estático compuesto puramente por HTML semántico y CSS, cumpliendo con los requisitos de accesibilidad y rendimiento.

## Descripción

Se trata de una página web responsiva donde presento mi perfil profesional, mi stack tecnológico enfocado en Full Stack (React, Node.js, TypeScript) y enlaces a evidencia técnica real de los sistemas que he desarrollado.

## Tecnologías utilizadas

- **Astro:** Framework utilizado como generador de sitio estático (SSG) para modularizar el código manteniendo un output final sin JavaScript innecesario.
- **Tailwind CSS:** Framework de utilidades para generar el archivo CSS externo y garantizar el diseño responsivo.
- **HTML5:** Estructura semántica (`header`, `main`, `section`, `article`).

## Funcionalidades

- Presentación de perfil y contacto.
- Lista de tecnologías clave.
- Sección de proyectos con enlaces directos al código fuente (Evidencia Técnica).
- Diseño completamente responsivo (Mobile First).
